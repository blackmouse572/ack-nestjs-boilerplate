## Key Features

### Authentication & Security

- JWT-based authentication with refresh tokens
- Social login (Google, Apple)
- Two-factor authentication (2FA/TOTP)
- API key authentication
- Session management with Redis
- Password policies and encryption

### Authorization & Access Control

- Role-based access control (RBAC)
- Policy-based authorization with CASL
- User, role, and permission management
- Term & policy acceptance tracking
- Fine-grained access control

### Database & Storage

- Prisma ORM with TypeScript
- Multi-database support (MongoDB, PostgreSQL, MySQL)
- AWS S3 integration for file storage
- Presigned URL generation
- Database migrations and seeding

### Development Experience

- TypeScript strict mode
- Comprehensive testing (unit, integration, e2e)
- Swagger/OpenAPI documentation
- Docker and docker-compose support
- Hot reload in development
- Code formatting and linting

## Getting Started

```bash
# Clone the repository
git clone https://github.com/andrechristikan/ack-nestjs-boilerplate

# Install dependencies
pnpm install

# Setup environment
cp .env.example .env

# Run with Docker
docker-compose up -d

# Access API documentation
open http://localhost:3000/docs
```

For detailed installation instructions, see the [Installation Guide](/installation).

## Architecture

The boilerplate follows best practices and design patterns:

- **Repository Pattern** - Clean data access abstraction
- **SOLID Principles** - Maintainable and testable code
- **Modular Architecture** - Feature-based organization
- **12-Factor App** - Cloud-native configuration
- **Domain-Driven Design** - Business logic separation
